System requirements


iPhoto Library Manager 3.5 requires Mac OS X 10.3.9 or later and will run on either a PPC or Intel based Macintosh. You can create, add, and switch between libraries, as well as use iPod Folders with any version of iPhoto. The album copying, library merging, library rebuild, and photo importing features all require iPhoto 4.0.3 or later.
